Тестовый интернет магазин содержит:

каталог товаров + фильтрация по категориям
корзина с возможностью изменения содержимого и оформление заказа
aвтоизация + регистрация

админская страница по пути http://localhost:8084/admin/ Нужно залогиниться под root root
Можно увеличивать права пользователю, создавать новую категорию товаров и новые карточки товара
и просматривать корзины


для работы нужно выполнить команды database.sql
