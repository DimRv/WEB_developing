compile- от англ. compile

Синтаксис

compile(source, filename, mode) -> code
source Строка. Текст кода который нужно выполнить. Обязателен.
filename Строка. Имя скрипта от куда берется код или "<string>" если source строка кода. Обязателен.
mode Строка. Обязателен. Один из вариантов:
  • 'eval' - одно выражение
  • 'exec' - блок операторов
  • 'single' - код из одного оператора

Описание

Встроенная функция compile (от англ. compile - компилировать) — преобразует код-строку в исполняемый код.
