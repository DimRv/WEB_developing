callable - от англ. callable

Синтаксис

callable(obj) -> bool
obj Объект любого типа данных. Обязателен.
bool True или False в зависимости может ли объект быть вызван

Описание

Встроенная функция callable (от англ. callable - вызываемый) — возвращает Тrue для объектов, которые можно вызвать, иначе False.

В вызываемых объектах реализован метод __call__, например это функции или классы.
