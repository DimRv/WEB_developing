eval - от англ. evaluate

Синтаксис

eval(code, globals=None, locals=None) -> object
code Строка-выражение python-кода или результат выполнения compile. Обязателен
globals Словарь из ключей и значений глобальных переменных участвующих в выражении
locals Словарь из ключей и значений локальных переменных участвующих в выражении
object Результат выполнения выражения или None, если выражение ничего не возвращает

Описание

Встроенная функция eval (от англ. evaluate - рассчитать) — выполняет код строки code как выражение python-кода и возвращает результат ее выполнения. Очень похожа на встроенную функцию exec, только выполняет одну строку кода.

Если code не является выражением, а например инструкция из нескольких строк кода, то возникнет исключение SyntaxError
