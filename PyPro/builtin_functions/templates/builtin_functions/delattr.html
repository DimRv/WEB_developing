delattr - от англ. delete attribute

Синтаксис

delattr(obj, attr_name) -> None
obj Объект в котором нужно удалить атрибут. Обязателен.
attr_name Строка. Название аттрибута для удаления. Обязателен.

Описание

Встроенная функция delattr (от англ. delete - удалить; attribute - атрибут) — удаляет атрибут в указанном объекте.

Аналог команды del obj.attr, только в качестве аргумента передается строка.
