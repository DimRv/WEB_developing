any - от англ. any

Синтаксис

any(iter) -> bool
iter Любой итерируемый объект, например, объекты типа list, tuple или set. Обязателен
bool True - если хотя бы один элемент итерируемого объекта в булевом виде True, иначе False

Описание

Встроенная функция any (от англ. any - любой) — возвращает True если хоть один элемент в объекте равен True в булевом представлении.

Если в параметре передан пустой итератор, то функция вернет False.

По своему функционалу очень похожа на функцию all, только та возвращает True если все элементы в объекте True.
