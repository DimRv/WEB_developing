ascii - от англ. American Standard Code for Information Interchange

Синтаксис

ascii(obj) -> str
obj Любой объект, например, объекты типа str, tuple или set. Обязателен
str Строка в которой все не ASCII-символы заменены их Unicode кодами

Описание

Встроенная функция ascii (от англ. American Standard Code for Information Interchange - Американский cтандарт кодирования для обмена информацией) — подобен функции str, только возвращаемая строка будет состоять из символов входящих в ACSII, остальные символы будут заменены их Unicode кодами.

ACSII это код один из первых стандартов кодирования символов для передачи текста по сети, используется с 1963 года и применяется до сих пор. Таблица символов состоит из 128 значений (27), состоящих из 32 команд управления телетайпом, символов английского языка, цифр и знаков.
