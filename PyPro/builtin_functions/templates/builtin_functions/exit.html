exit - от англ. exit

Синтаксис

exit() -> None
None Объект ничего

Описание

Встроенная функция exit (от англ. exit - выход) — завершает выполнение программы. Функция обычно используется для завершения работы интерпретатора. В скрипах используется похожая функция sys.exit(), которая пом возвращает операционной системе код завершения программы.
