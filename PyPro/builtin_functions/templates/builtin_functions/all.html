all - от англ. all

Синтаксис

all(iter) -> bool
iter Любой итерируемый объект, например, объекты типа list, tuple или set. Обязателен
bool True - если все элементы итерируемого объекта в булевом виде True, иначе False

Описание

Встроенная функция all (от англ. all - все) — возвращает True если все элементы, переданного в параметре итерируемого объекта возвращают True в булевом представлении.

Если в параметре передан пустой итератор, то функция вернет True.

По своему функционалу очень похожа на функцию any, только та возвращает True если хотя бы один элемент в объекте True.
