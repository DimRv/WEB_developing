chr - от англ. char

Синтаксис

chr(int) -> str
int Число типа int от 0 до 1114111. Обязателен.
str Символ, полученный из Unicode таблицы символов

Описание

Встроенная функция chr (от англ. character - символ) — Возвращает строку, символ из Unicode таблицы по ее позиции.

Целое число, переданное в параметре может быть от 0 до 1114111 включительно. Если число не входит в диапазон но возникнет исключение ValueError
