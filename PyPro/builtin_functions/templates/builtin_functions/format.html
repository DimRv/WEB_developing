format - от англ. format

Синтаксис

format(obj, formatting) -> str
obj Объект, который нужно форматировать, обычно это строки, числа или дата. Обязателен
formatting Строка которая описывает как должно происходить форматирование.

Описание

Встроенная функция format (от англ. format - форматировать) — предназначена для преобразования к строке объекта obj путем использования метода __format__ класса объекта obj: type(obj).__format__().

Параметр formatting - строка-схема, которая передается методу __format__ и определяет как будет выглядеть возвращаемая строка. Для встроенных в python типы данная строка обычно представлена в так называемым Format Specification Mini Language (Мини язык описывающий форматирование)

Format Specification Mini Language

Строку описывающую форматирование обычно представляют в следующем виде, но сработает параметр или нет зависит от типа передаваемого объекта, например, 'z' работает только с float числами.

[[fill]align][sign]["z"]["#"]["0"][width][grouping_option]["." precision][type]

Описание этой структуры лучше начать с конца:

[type]

Определяет один из типов преобразования:

  • b - преобразует целое число к двоичной системе исчисления. (для int)
  • c - преобразует целое число как символ Unicode. (для int)
  • d - преобразует целое число к десятичной системе исчисления. (для int)
  • e - преобразует число к так называемому научному представлению с символом e.
  • E - преобразует число к так называемому научному представлению с символом E.
  • f - преобразует число к фиксированному кол-ву символом после точки.
  • F - как f, только 'nan' станет 'NAN', a 'inf' в 'INF'
  • g - общий формат представления числа, округляет число и приводит к типу 'f' или 'e' в зависимости от кол-ва символов.
  • G - тоже что и g, но использует 'E' преобразование вместо 'e'
  • n - аналог d, то использует настройки локали для определения символа разделителя больших чисел.
  • o - преобразует целое число к восьмеричной системе исчисления. (для int)
  • s - преобразует к строке. (для строк)
  • x - преобразует целое число к шестнадцатерично системе исчисления с малыми символами для чисел выше 9. (для int)
  • X - преобразует целое число к шестнадцатерично системе исчисления с заглавными символами для чисел выше 9. (для int)
  • % - процентный формат, умножает на 100 и преобразует к 'f' добавляя символ '%'
["." precision] Указывает точность представления числа - кол-во символов после точки путем округления round.
[grouping_option]

Указывает символ для разделения больших чисел

  • _ — тысячи разделяются символом '_'
  • , — тысячи разделяются символом ','
[width] Минимальный размер строки, включая префиксы, разделители. Если не указан, то размер определится по содержимому.
[0] Устарело с версии 3.11. Ранее добавление нуля перед width означало [fill]:0; [alignment]: "="
[#] Добавляет '0b', '0o', '0x', '0X' к финальному преобразованию для типов 'b', 'o', 'х', 'X'. Для вещественных чисел наличие плавающей точки становится обязательным.
[z] Убирает знак отрицания для чисел чье значение округлилось к нулю после ["." precision].
[sign]

Определяет нужно ли отображать знак у чисел

  • + — знак числа будет всегда отображаться и для положительных и для отрицательных значений.
  • - — знак отображается только у отрицательных значений. Стандартное поведение.
  • space — оставляет место для положительных чисел, минус остается для отрицательных.
[[fill]align]

Параметры будут иметь эффект, только если [width] будет дольше длины строки.

[fill] - определяет каким знаком заполнятся недостающие значения. По умолчанию пробелом.

[align] - определяет где будет располагаться текст в доступном пространстве, возможны значения:

  • < — текст слева.
  • > — текст справа.
  • ^ — текст выравнивается по центру.
  • = — заполняет доступное пространство за знаком числа.
