{{title}}

dict - от англ. dictionary

Синтаксис

dict() -> dict
dict(kwargs) -> dict
kwargs Итерируемый объект, содержащий с себе кортеж (k, v), где k станет ключом словаря, v - значением.
dict Возвращаемый словарь.

Описание

Встроенная функция dict (от англ. dictionary - словарь) — предназначен для создания словаря с помощью итерируемого объекта, содержащего кортеж двух значений: (ключ, значение присваиваемое ключу)

Если используется без параметров, то возвращает пустой словарь.
