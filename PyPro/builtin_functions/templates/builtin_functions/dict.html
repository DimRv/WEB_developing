dict - от англ. dictionary

Синтаксис

dict() -> dict
dict(mapping) -> dict
dict(iterable) -> dict
dict(**kwargs) -> dict
mapping Объект, в котором реализован метод keys() и получение объекта __getitem__
iterable Итерируемый объект, элементы которого кортежи в формате (ключ, значение)
**kwargs Параметры в формате параметр=значение
dict Словарь

Описание

Встроенная функция dict (от англ. dictionary - словарь) — предназначена для создания нового объекта словаря.

Если используется без параметров, то возвращает пустой словарь.
