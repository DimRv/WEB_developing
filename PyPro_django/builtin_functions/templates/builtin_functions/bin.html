bin - от англ. binary

Синтаксис

bin(int) -> str
int Число типа int. Обязателен.
str Строка, представление числа в двоичном виде

Описание

Встроенная функция bin (от англ. binary - бинарный) — Возвращает строку в формате 0b + двоичное представление числа.
