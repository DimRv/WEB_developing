bool - от англ. boolean

Синтаксис

bool(obj) -> bool
obj Объект любого типа данных. Обязателен.
bool Булево представление объекта

Описание

Встроенная функция bool (от англ. boolean - булевый) — возвращает False для пустых итерируемых объектов, 0 и для None, для всех остальных объектов вернет True. Подробнее о типе bool.

Вызывает метод __bool__ объекта, переданного в параметре.
