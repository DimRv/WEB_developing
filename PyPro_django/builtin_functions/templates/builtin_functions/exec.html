{{title}}

exec - от англ. execute

Синтаксис

exec(code, globals=None, locals=None, closure=None) -> object
code Строка содержащая python-код, объект-код или результат выполнения compile. Обязателен
globals Словарь из ключей и значений глобальных переменных участвующих в выражении
locals Словарь из ключей и значений локальных переменных участвующих в выражении
closure Кортеж из Cell-объектов, используется для работы замыканий
object Результат выполнения code или None, если code ничего не возвращает

Описание

Встроенная функция exec (от англ. execute - выполнить) — выполняет инструкции python-кода переданные в code и возвращает результат их выполнения. Очень похожа на встроенную функцию eval, только может выполнять многострочные инструкции
