bytearray - от англ. byte; array

Синтаксис

bytearray() -> bytearray
bytearray(int) -> bytearray
bytearray(iter) -> bytearray
bytearray(str, encoding, error) -> bytearray
int Число типа int.
iter Итератор, элементы которого целые числа int.
str Строка str.
encoding Кодировка для перевода символов, например "ascii" или "utf-8". Обязателен если первый параметр str.
error Параметр указывает как обрабатывать ошибки, полученные при кодировке строки.

Возможные значения:

  • 'strict' - создается исключение UnicodeEncodeError
  • 'ignore' - отсеивает ошибочные символы
  • 'replace' - заменяет ошибочные символы на знак ?
  • 'xmlcharrefreplace' - заменяет на формат &#код;
  • 'backslashreplace' - заменяет на формат \\uкод
  • 'namereplace' - заменяет на формат \\N{Описание Кода Символа}
bytearray Объект типа изменяемая байтовая строка.

Описание

Встроенная функция bytearray (от англ. byte - байт; array - массив) — используется для создания или преобразования объектов к типу не изменяемая байтовая строка

Создает пустую не изменяемую байтовую строку, если в параметре ничего не было передано.

Если в параметре было указано целое число x, то создается не изменяемая байтовая строка длины x

В итераторе может быть любая последовательность из целых чисел от 0 до 255.

Если необходимо создать не изменяемую байтовую строку из обычной строки, то обязательно нужно указать кодировку для преобразования в encoding
