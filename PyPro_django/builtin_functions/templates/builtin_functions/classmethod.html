classmethod - от англ. class; method

Синтаксис

classmethod(func) -> method
func Функция, которая содержит в параметре объект класса. Обязателен.
method Метод, предназначенный для взаимодействия с объектом класса, а не с экземплярами класса.

Описание

Встроенная функция classmethod (от англ. class - класс, method - метод) — преобразует переданную в параметре функцию в метод класса.

Метод - функция объявленная в теле класса, содержит первым параметром self - ссылку на экземпляр, предназначена для взаимодействия с экземплярами класса.

Метод класса - метод который содержит первым параметром cls - ссылку на класс, предназначен для взаимодействия с классом экземпляра, с его атрибутами и методами.
