dir - от англ. directory

Синтаксис

dir() -> list
dir(object) -> list
object Объект для которого нужно получить список доступных атрибутов.
list Список строк

Описание

Встроенная функция dir (от англ. directory - директория) — возвращает список аргументов доступных объекту. Данный список может быть получен из __dir__ метода объекта или путем получения __dict__.keys() самого объекта и всех доступных объектов в иерархии наследования. Список может быть не полным при наличии в объекте __getattr__.

Если в качестве параметра передан модуль, то функция вернет список всех атрибутов данного модуля.

Если используется без параметров, то возвращает список переменных в текущей области видимости.
