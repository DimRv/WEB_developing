breakpoint - от англ. break point

Синтаксис

breakpoint() -> None

Описание

Встроенная функция breakpoint (от англ. break - разрыв; point - точка) — останавливает выполнение программы и запускает отладчик с помощью которого можно разобраться в ошибках программы. Отладчик по умолчанию — PDB. В момент останова переменные содержат значения на текущую строку выполнения кода.

Для выхода из режима отладки и продолжения выполнения кода необходимо выполнить команду 'c' (от англ. continue - продолжить). Нажать с+Enter.

Точки останова могут быть проигнорированы если в переменную окружения PYTHONBREAKPOINT установить 0.

Через переменную окружения PYTHONBREAKPOINT также можно задать отладчик, указав необходимую функцию.
