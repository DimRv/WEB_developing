filter - от англ. filter

Синтаксис

filter(None, iterable) -> filter-obj
filter(function, iterable) -> filter-obj
None Объект ничего
iterable Итерируемый объект
function Функция которая возвращает True или False для каждого элемента итератора
filter-obj Объект фильтр, при каждом обращении возвращает следующий элемент итерируемого объекта для которого function вернула True

Описание

Встроенная функция filter (от англ. filter - фильтровать) — предназначена для создания нового итерируемого объекта путем отсева элементов в исходном итерируемом объекте при применении генераторной-функции. Функция принимает в качестве параметра элемент массива, и если та возвращает True, то данный элемент окажется в результирующем объекте, False же уберет элемент из вывода.

Если вместо функции используется None, то отфильтруются те элементы для которых функция bool вернет False.

Исходный итерируемый объект не меняется.
