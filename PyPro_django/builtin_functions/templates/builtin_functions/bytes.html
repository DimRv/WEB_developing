bytes - от англ. bytes

Синтаксис

bytes() -> bytes
bytes(int) -> bytes
bytes(iter) -> bytes
bytes(str, encoding, error) -> bytes
int Число типа int.
iter Итератор, элементы которого целые числа int.
str Строка str.
encoding Кодировка для перевода символов, например "ascii" или "utf-8". Обязателен если первый параметр str.
error Параметр указывает как обрабатывать ошибки, полученные при кодировке строки.

Возможные значения:

  • 'strict' - создается исключение UnicodeEncodeError
  • 'ignore' - отсеивает ошибочные символы
  • 'replace' - заменяет ошибочные символы на знак ?
  • 'xmlcharrefreplace' - заменяет на формат &#код;
  • 'backslashreplace' - заменяет на формат \\uкод
  • 'namereplace' - заменяет на формат \\N{Описание Кода Символа}
bytes Объект типа байтовая строка.

Описание

Встроенная функция bytes (от англ. bytes - байты) — используется для создания или преобразования объектов к типу байтовая строка.

Содержимое созданной байтовой строки зависит от типа объекта, который бы передан в параметре:
